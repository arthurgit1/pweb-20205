O que é servidor web e o que é servidor de aplicação? Qual a diferença?

Um servidor WEB hospeda um serviço que pode ser acessado pelo seu navegador. ... Este servidor é WEB. Servidor de aplicação é aquele que hospeda um software. Por exemplo, imagine o ERP SAP.

Fale um pouco sobre um framework web e um framework ORM da linguagem escolhida.

Framework é uma abstração que une códigos comuns entre vários projetos de software provendo uma funcionalidade genérica. Um framework pode atingir uma funcionalidade específica, por configuração, durante a programação de uma aplicação.
